终于搭建好了这个博客，记录一下安装过程

> * （必须）安装ruby,Devkit
> * （必须）安装Jekyll
> * （非必须）Python，


其中安装Jekyll的时候，我总是遇到问题，结果发现问题是天朝的网络在作祟
先运行如下代码即可

{% highlight ruby %}
$ gem sources --remove https://rubygems.org/
$ gem sources -a https://ruby.taobao.org/
$ gem sources -l
*** CURRENT SOURCES ***

https://ruby.taobao.org
# 请确保只有 ruby.taobao.org
$ gem install rails
{% endhighlight %}
